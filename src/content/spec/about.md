# 关于我 / About Me

## 🛠️ 关于本站

这个网站使用 **Astro** 框架构建，采用了 [Firefly](https://github.com/CuteLeaf/Firefly) 主题模板。Firefly 是一款清新美观的 Astro 静态博客主题，基于 [Fuwari](https://github.com/saicaca/fuwari) 二次开发。

在这里，我会分享：

- 💻 **技术文章**：Linux、Docker、编程开发
- 🎵 **音乐分享**：喜欢的歌曲和乐队
- 🎬 **ACG**：动漫、游戏相关内容
- ✨ **日常随笔**：生活中的思考和感悟
